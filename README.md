# ResQSense
> AI-powered disaster awareness, eco-risk scoring, and personal safety platform built for India.
>
> #how it works
> Live Disaster Alerts — Real-time earthquake, flood, and cyclone data using USGS & ReliefWeb APIs
> Weather Risk — Current weather conditions using OpenWeatherMap API
> AI Safety Advice — Personalized safety guidance powered by Claude AI
> Eco-Risk Score — Quiz that connects your daily habits to environmental risk
> Emergency Kit Builder — AI-generated emergency kit based on your city's risk profile

#

- Vanilla JavaScript (ES6+, Fetch API, async/await)

- Maps: Leaflet.js
